// we will use the variable const to define the following data types //

                
const name = 'Kenji'; // this is a string //

                
const isCool = true; // this is a boolean //

                
const x = null; // this is null //

                
const y = undefined; // this is undefined// 

                
const age = '18'; // this is a number //

              
            

            

              to display the type of data type in the console, you can type the following;
            

            

              
                

                
console.log(typeof name); // this will display 'string' in the console //

              
            


            

              You can do alot with string data types, like; concatenation,
              template string, template string as a variable, number of
              characters in a string
            


            

              
// concatenation //

                
console.log('Howdy' + name + ', how are you?');

                

                
// template string //

                
console.log(`your name is ${name} and you are ${age} years old`);

                

                
// as a variable //

                
const convo = `your name is ${name} and you are ${age} years old`;

                
console.log(convo);

                

                
// number of characters in a string //

                
const s = 'hello world!';

                
console.log(s.length);

              
            


            

              Arrays
            

            

              What is an array?
            

            

              An array is a list of items contained within a variable
            

            

              
// for example //

                
const animals = ['duck', 'antelope', 'kiwi', 'whale'];

                
console.log(animals);

              
            

            

              Now, arrays are 'zero based' which means that the first value in the array is 0 and so on and so forth. If we want to display a specfic positon or value we could do the following;
            

            

              
// lets display 'kiwi' from the animal array //

                
console.log(animals[2]);

              
            

            

              We can also add a value to the function but we cannot reassign since we are using const
            

            

              
// lets add 'bear' to the animal array //

                
animals[4] = 'bear';

                
console.log(animals);

              
            

            

              The following are some other ways to modify an array
            

            

              
// .push add value to the end //

                
animals.push('tiger');

                
console.log(animals);

                
 

                
// .unshift adds value to the front //

                
animals.unshift('shrew');

                
console.log(animals);

                
 

                
// .pop takes the last value off //

                
animals.pop();

                
console.log(animals);
