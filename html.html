Elements and Tags

What is an element?

An element is composed of a tag and its content. Each element has a specific meaning that the browser recognizes, which means HTML is a semantic language

opening tag <p>
the content lorem ipsum
closing tag </p>

Empty elements

Most elements have an opening and closing tag but... empty elements only have one tag with no contents (exception: image tag has a link inside)

Horizontal rule <hr/>
line break <br/>
image <img/>

Required HTML elements for a web page 

<!DOCTYPE html>
  <html>
    <head>
    </head>
    <body>
    "Hello world!"
    </body>
  </html>

Block and inline elements

Elements can either be block or inline. Block takes up the entire width of the web page and inline only occupy space inside of a block element
