What is the css rule set?

selector ex. paragraph (p)
property ex. color
property value ex. lightblue

together, the css rule set makes what is called a declaration, which is observed in the code snippet below 

p {
  color: lightblue;
}

you can apply a css rule set to mulitple elements by using a comma between the selectors 

h2, h3, p {
  color: lightblue;
}

Basic properties

Fonts

All of these declarations allow you to style text to your liking 

p {
  font-family: 'Courier new', sans-serif;
  font-size: 18px;
  font-weight: 400;
  font-style: italic;
}

To make life eassier, this can be consensed to one declartion. 

p {
  font: 'Courier new', italic, 18px, 400 sans-serif;
}

Coloring 

/* This changes the text color */
p {
  color: rebeccapurple;
}

                
/* This changes the background color */

                
p {

                
  background-color: aliceblue;

              Positioning
            

            

              Positioning is the key to a layout on your web page. This property allows you to move elements around where ever you like
            

            

              

                static
                this is the default positioning of the element
              

              

                relative
                
                  changes the positioning in relation to where it was originally
                
              

              

                absolute
                
                  changes the positioning in relation to the dimensions of the
                  webpage
                
              

              

                sticky
                
              

              

                fixed
                
              

            


            

              for relative and absolute positioning, you can use the top, right,
              bottom and left attributes to specify
            

            

              

                top: 50px;
                will position the element 50px from the top
              

              

                right: 50px;
                
                  will position the element 50px from the right
                
              

              

                bottom: 50px;
                
                  will position the element 50px from the bottom
                
              

              

                left: 50px;
                will position the element 50px from the left
              

            


            

              here are the following declarations;
            


            

              
                
img {

                
  position: static;

                
}

                

                
img {

                
  position: relative;

                
}

                

                
img {

                
  position: absolute;

                
}

                

                
img {

                
  position: sticky;

                
}

                

                
img {

                
  position: fixed;

                
}
